Launch

Newt is a free web based, open source software that is expected to work on most major browsers with recent releases without any prior installation or set up.

Architecture

Newt is based on and is a sample application for the customizable JavaScript library named ChiSE.js, which in turn is based on a JavaScript library named SBGNViz.js.
